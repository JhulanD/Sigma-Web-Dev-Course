CSS Transforms MasterClass Demo

rotate(20deg)
rotate(angle): Rotates the element clockwise or counterclockwise by the specified angle. Positive angles rotate clockwise.
transform: rotate(20deg);
scale(1.2)
scale(factor): Enlarges or shrinks the element by the scale factor. A value over 1 makes it bigger, less than 1 smaller.
transform: scale(1.2);
skewY(20deg)
skew(axis): Tilts the element along the X or Y axis, creating a slanted effect like shearing.
transform: skewY(20deg);
translateY(-30px)
translate(x, y): Moves the element from its current position horizontally (x) or vertically (y).
transform: translateY(-30px);
perspective + rotateY(30deg)
perspective(value): Gives a 3D depth effect, making rotations in 3D space look realistic.
transform: perspective(200px) rotateY(30deg);
matrix(1.1, 0.2, 0.2, 1.1, 0, 0)
matrix(a,b,c,d,e,f): Combines multiple 2D transforms (scale, skew, rotate, translate) into one function using six values.
transform: matrix(1.1, 0.2, 0.2, 1.1, 0, 0);
scaleX(-1)
scaleX(-1): Flips the element horizontally, creating a mirror image along the vertical axis.
transform: scaleX(-1);
